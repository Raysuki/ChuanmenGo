#  串门宝(ChuanMenGo) - 智治北疆网格员工作台

## 🌟 核心功能

### 1. 数字化邻里册 (Household Management)
- **精细化建档**：记录村民基本信息、联系方式、家庭成员及特长。
- **风险分级管理**：根据村民情况（如独居老人、残疾人等）设置红、黄、绿三色风险等级，自动生成走访提醒。
- **特长标签**：挖掘乡村能人，方便开展互助活动。

### 2. 智能走访签到 (Visit Tracking)
- **待办提醒**：根据风险等级自动计算下次走访时间，确保“重点人群不漏访”。
- **现场实录**：支持文字记录身体状况、困难诉求，并支持**拍照上传**现场凭证。
- **历史溯源**：完整保留每一户的走访轨迹。

### 3. 积分治理体系 (Credit System)
- **正向激励**：针对环境卫生、志愿服务、邻里互助等行为发放积分。
- **拍照存证**：积分发放支持上传照片，确保治理过程公开、公平、公正。
- **积分汇总**：自动生成村民积分排行榜，激发参与热情。

### 4. 数据中心与导出 (Data Center)
- **Excel 深度集成**：支持一键导出邻里册、走访记录和积分明细。
- **照片同步导出**：**特色功能**，导出的 Excel 文件中自动嵌入现场实拍照片，方便存档与汇报。
- **离线持久化**：数据存储于本地浏览器，无需复杂后端即可实现即开即用。

### 5. 随手记 (Notes)
- 方便记录走访过程中的临时想法、待办事项或政策要点。

## 🛠️ 技术栈

- **前端框架**: React 19 + TypeScript
- **样式处理**: Tailwind CSS (现代、响应式设计)
- **动画效果**: Framer Motion (流畅的移动端交互)
- **图标库**: Lucide React
- **数据处理**: 
  - `xlsx`: 基础 Excel 解析
  - `exceljs`: 实现带图片的复杂 Excel 导出
- **存储方案**: LocalStorage (本地持久化)
- **AI 能力**: Google Gemini API (预留智能分析接口)
